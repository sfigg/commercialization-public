# CustomScancodes


Blocks the list of custom scan codes.

## Type


DataTypeEnum

## Values


An enumeration containing one or more [WEKF\_Scancode](https://docs.microsoft.com/en-us/windows-hardware/customize/enterprise/wekf-scancode) values.
