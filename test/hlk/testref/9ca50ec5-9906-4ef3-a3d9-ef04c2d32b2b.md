# <span id="p_hlk_test.9ca50ec5-9906-4ef3-a3d9-ef04c2d32b2b"></span>WDDM2 - Page Fault Test - Induce Fault 9 Test


Verifies that the hardware and DX9 driver support and can recover from page faults caused by accessing invalid GPU virtual addresses.
