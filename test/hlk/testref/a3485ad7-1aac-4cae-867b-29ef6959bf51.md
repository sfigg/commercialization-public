# <span id="p_hlk_test.a3485ad7-1aac-4cae-867b-29ef6959bf51"></span>Dynamic Resources


This automated test creates dynamic resources (vertex buffers, index buffers, and textures), and uses them with various lock flags.

This test renders an image, and compares pixels in the source image to an image that the Microsoft® Direct3D® reference (software rasterizer) creates. The test uses the comparison to verify that none of the buffers were dropped while locked.

This topic applies to the following test jobs:

-   Dynamic Resources

-   Dynamic Resources (WoW64)
