# <span id="p_hlk_test.1c480aee-ac2a-4893-9a74-3c317a05a2fe"></span>Camera Driver System Test - MediaCapture - TestExposureControl


This test validates the camera's exposure control in supported settings and conditions. If supported, validates ability to get auto, min, max, step values; that the value range (min, max, step) is correct; can set auto and in-range values; cannot set out-of-range values.
