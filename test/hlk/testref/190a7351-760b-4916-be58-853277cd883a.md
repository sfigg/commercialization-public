# <span id="p_hlk_test.190a7351-760b-4916-be58-853277cd883a"></span>Camera Driver System Test - MediaCapture - ValidateFrameFocusValues


This test validates Focus values exposed by the camera via variable photo sequence capture (3 frame and 10 frame). In each case it will step through the exposed range of values based on the number of frames captured.
