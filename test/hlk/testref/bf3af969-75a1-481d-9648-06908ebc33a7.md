# <span id="p_hlk_test.bf3af969-75a1-481d-9648-06908ebc33a7"></span>Win6\_4.MB.CDMA.Data.TestSignalLoss


The test verifies the device provides status indications in the right sequence when signal strength loss occurs when packet service was attached.
