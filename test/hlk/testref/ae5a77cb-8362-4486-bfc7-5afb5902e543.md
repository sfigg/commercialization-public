## More Information
### Parameters
| Parameter Name | Parameter Description |
| --- | --- |
| **SarMode** | Specifies SAR control mode [Device|OS] |
| **SarBackoffStatus** | Enable or disable SAR backoff [Enable|Disable] |
| **SarConfigurations** | Specifies SAR backoff index per antenna. Format: AntennaIndex:BackoffIndex[,AntennaIndex:BackoffIndex,...] Example: 0:2,3:1 |
| **TransmissionStatusNotification** | Enable or disable channel transmission status notification [Enable|Disable] |
| **HysteresisTimer** | Specifies hysteresis timer. Range: 1 - 5 |
