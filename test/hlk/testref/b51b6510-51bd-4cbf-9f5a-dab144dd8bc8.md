# <span id="p_hlk_test.b51b6510-51bd-4cbf-9f5a-dab144dd8bc8"></span>Camera Driver System Test - MediaCapture - ValidateZoomControlSupportedScenarios


This test validates the camera's Zoom control in supported settings and conditions. If zoom is supported, validates that advertised min, max, step values are correct; zoom can be set with in-range values; zoom cannot be set with out-of-range values.
