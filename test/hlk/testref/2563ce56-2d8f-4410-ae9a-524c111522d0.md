# <span id="p_hlk_test.2563ce56-2d8f-4410-ae9a-524c111522d0"></span>WDDM - OfferReclaim11 - Decommit Repurposed Test


Performs a non-lazy full trim pass to decommit all trimmable memory in the process which has already been repurposed by the NT memory manager.
