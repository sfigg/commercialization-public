# <span id="p_hlk_test.d50e1195-b476-42d1-96b8-1d2241cc1d38"></span>System Audio - MFCaptureEngine API Test - Capture MP3 Audio samples from the default capture pin


This test validates that the selected audio capture endpoint is compatible with Windows Media Foundation APIs (MFCaptureEngine)
