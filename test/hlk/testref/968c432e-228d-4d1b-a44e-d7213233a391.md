# <span id="p_hlk_test.968c432e-228d-4d1b-a44e-d7213233a391"></span>Test.ReportRate


This test verifies that the device reports at the required rate during single finger interactions.

## <span id="Running_the_test"></span><span id="running_the_test"></span><span id="RUNNING_THE_TEST"></span>Running the test


Before you run the test, complete the test setup as described in the test requirements: [Mouse or other Pointing Device Testing Prerequisites](mouse-or-other-pointing-device-testing-prerequisites.md).

1.  Following the instructions on the screen, move one contact around on the touchpad for 1, 5, or 20 seconds. (a clock in the upper right corner shows how many seconds have passed since all contacts have been down).

2.  Repeat 11 times. Note that the length of time will change for some iterations, so please read and follow the on-screen instructions for each iteration

    PTLogo will fail an iteration and display an error if there is a problem with the report rate. Otherwise, the iteration will pass upon lift, and PTLogo will advance to the next iteration.
