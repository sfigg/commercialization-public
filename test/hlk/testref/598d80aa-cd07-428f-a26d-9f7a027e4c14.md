# SW-DRM Video - Play protected content using software decryptor

>[!IMPORTANT]
>  
This test requires supplemental content. You can find this content in the 'Windows HLK supplemental test content' section at the following location:

-   <http://go.microsoft.com/fwlink/p/?LinkId=532718>

 

Verifies if windows runtime app can play protected video content that requires Output Protection Management(OPM)
