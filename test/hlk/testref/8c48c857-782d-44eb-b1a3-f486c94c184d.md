# <span id="p_hlk_test.8c48c857-782d-44eb-b1a3-f486c94c184d"></span>Camera Driver System Test - MediaCapture - ValidateSceneModeControlPresets


This test validates that all Scene Mode preset values exposed or permitted by the camera work properly. The test will attempt to take a photo after setting the Scene Mode.
