# <span id="p_hlk_test.822e5538-dabd-478f-a121-2601d8981157"></span>D3D12 - Resource Binding - ScatterGather copy descriptors from one heap into another


Validates whether descriptor copies work correctly, by creating an arbitrary descriptor heap with descriptor entries reversing this heap into an intermediate heap and reconstructing the original heap and rendering a rectangle which uses all heap entries to produce a color.
