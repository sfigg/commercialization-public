# <span id="p_hlk_test.23ddaa2b-110a-4b61-90da-62b97818281d"></span>WDDM LockConfirm


This automated test executes different locking scenarios and verifies that surface pattern remains the same after the lock.

This topic applies to the following test jobs:

-   WDDM LockConfirm

-   WDDM LockConfirm (WoW64)

## Test details
|||
|---|---|
| **Specifications**  | <ul><li>Device.Graphics.WDDM.DisplayRender.Base</li></ul> |  
| **Platforms**   | <ul><li>Windows 10, client editions (x86)</li><li>Windows 10, client editions (x64)</li><li>Windows Server 2016 (x64)</li><li>Windows 10, client editions (ARM64)</li></ul> |

|**Requires special configuration**| false |
|**Type**| automatic |
