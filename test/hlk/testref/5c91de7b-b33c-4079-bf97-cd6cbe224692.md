# <span id="p_hlk_test.5c91de7b-b33c-4079-bf97-cd6cbe224692"></span>D3D12 - Static Samplers Limit Stress


Stress 2032 static sampler limit, since some drivers need to emulate these via hidden space in sampler descriptor heap(s).
