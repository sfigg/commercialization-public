# <span id="p_hlk_test.8cc609d2-954a-441b-9edd-78682273acb6"></span>Audio Codec - MFCaptureEngine API Test - Capture MP3 Audio samples from the default capture pin


This test validates that the selected audio capture endpoint is compatible with Windows Media Foundation APIs (MFCaptureEngine)
